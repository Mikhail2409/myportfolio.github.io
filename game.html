Quelle est la probabilité de trouver 5 bons numéros parmi 49?

Selon Wikipédia, pour trouver 5 bons numéros parmi 49 proposés, il y a une chance sur 2 118 760. Loto

Cette probabilité est extrêmement faible.

Découvrez des chances de gagner plus élevées avec 3 cartes "diamant" parmi 9 !

Pour calculer la probabilité de trouver 3 bonnes cartes parmi 9, nous utilisons la formule de probabilité combinatoire:

P(n, k) = (n!)/((k!)(n-k)!)

Dans ce cas, n = 9 (nombre total de cartes possibles) et k = 3 (nombre de cartes que nous voulons trouver).

P(9, 3) = (9!)/((3!)(9-3)!) = (9!)/((3!)(6!))

Calculons maintenant le résultat:

P(9, 3) = (9 x 8 x 7 x 6!)/((3 x 2 x 1)(6!)) = (9 x 8 x 7)/(3 x 2 x 1) = 504 / 6 = 84

Donc, la probabilité de trouver 3 bonnes cartes parmi 9 est de 1 sur 84.
